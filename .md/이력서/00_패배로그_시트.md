## 사인 클래스 (패배 분류 — 오분류 주의)

- **A. 필터 벽**: 학위·경력 요건 미달로 포폴을 열어보기 전에 끝난 것. 실력 문제 아님 → 지원 풀 조정으로 대응
- **B. 포폴 접근성**: 열었지만 30초 안에 강점이 안 읽힌 것 → 영상·README·첫인상 개선
- **C. 경험 서사**: 읽었는데 안 뽑힘 → 합격자 포폴과 대조해 서사 재구성
- **D. 코테**: 코딩테스트 탈락 → 알고리즘 트랙 천장 훈련 (못 푸는 티어)
- **E. 면접**: 면접 탈락 → 직후 전 질문 복기 + 합격자 답변 해부
