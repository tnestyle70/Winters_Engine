## 1. 차기 세션 실행 순서 (이대로만)

1. **영상**: 사용자가 촬영한 풀 영상 → 유튜브 업로드 1회
2. **GIF 컷**: 사용자가 "몇 초~몇 초 = 어느 섹션" 목록 제공 → `ffmpeg -ss <시작> -t 8 -i 풀영상.mp4 -vf "fps=15,scale=960:-1" out.gif` (슬롯 8개, 노션 무료 5MB/파일 — 넘치면 fps 12·scale 720)
3. **Notion 반영**: `SUBMIT_FINAL.md`(§2) 섹션별 붙여넣기 + 깨진 GIF 슬롯은 블록 메뉴 "바꾸기"로 교체 + 유튜브는 `?t=초` 타임스탬프 임베드(시작점만 지정 가능, 종료점 불가 — 정상)
4. **30초 게이트**: 각 섹션 도입 문단을 안 보고 30초 말하기 — 막히는 섹션만 ▶깊이 복습
5. **제출**: 펄어비스 인재풀(합불 통보 없음 = 부담 제로, 자체 엔진 회사라 정면 적합) + 스마일게이트 CROSSFIRE(자기소개 첫 줄 회사명 슬롯만 교체)
