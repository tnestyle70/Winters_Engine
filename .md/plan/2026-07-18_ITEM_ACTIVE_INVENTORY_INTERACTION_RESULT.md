## 갱신된 트레이드오프

- 이번 범위는 실제 입력 키와 HUD가 일치하는 6슬롯으로 고정했다. 8슬롯은 입력·레이아웃 규칙이 추가로 필요하므로 넣지 않았다.
- 갈라진 하늘 재사용 대기시간은 아이템 소유자 기준 단일 5초다. 대상별 재사용 대기시간은 별도 런타임 테이블이 필요하다.
- 존야 황금 외형은 서버의 독립 `StasisVisual` 상태를 복제해 클라이언트 재질 오버라이드로 표현한다. 이미 다른 시스템이 소유한 재질 오버라이드는 덮어쓴 뒤 무조건 초기화하지 않는다.
- 와드 충전 수, 액티브 쿨다운 숫자/오버레이, 헤르메스 시미터의 이동 속도 후속 효과는 이번 요구 범위에 포함하지 않았다.
- 서버 권위 로직과 클라이언트 빌드는 자동 검증으로 닫았지만, 실제 창에서 드래그 감각·미니맵 가림·황금 재질을 눈으로 확인하는 시각 스모크는 수동 확인 항목으로 남는다.
