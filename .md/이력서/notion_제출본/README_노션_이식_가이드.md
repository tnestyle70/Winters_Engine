## 구조 결정 (확정)

- **Winters만 김태호식 딥다이브, 나머지는 독립 카드.** 전부 Winters 밑에 넣지 않는 이유: ①딥다이브의 힘은 "하나를 끝까지"인데 4개를 품으면 묽어진다 ②협업(던전스)·기본기(스타)·완성(Liberation)·문제해결(NYPC)은 면접관이 따로 찾는 서로 다른 리스크 해소 증거다 ③"하나로 이어지는 느낌"은 홈의 성장 경로 스트립 + 각 카드 끝의 "Winters로 이어진 것" 문장이 담당한다.
- 페이지 순서 = 깊이(1) → 협업(2) → 지능(3) → 기본기(4) → 완성(5) → 방식(6) → 사람(7).
