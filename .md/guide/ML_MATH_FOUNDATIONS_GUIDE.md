Session - LLM/RL/IL을 떠받치는 수학을 기초부터 적층하는 학습 트랙을 고정한다 (게임 수학 → ML 수학 다리).

작성일: 2026-07-12
성격: 학습 가이드 (FIBER_LEARNING_GUIDE 패턴). 코드 계획서 아님.
운영 규칙 (GOAL_OPERATING_DOCTRINE): **모듈 환전 게이트** — 각 모듈은 (a) 실행 코드 1개, (b) 공개 기술 문답 후보 3개+, (c) Winters 적용 연결 1개를 산출해야 다음 모듈 진입. 외부 마감 = RL 직무 기술면접 (이 수학이 출제 범위다). 주간 30% 천장 예산은 이 트랙이 아니라 제출·공개물에 쓴다.

## 설계 원리

- **아는 것에서 출발**: 사용자는 게임 수학(벡터/행렬/변환), 결정론 시뮬, 골드 가치화, NYPC 측정 규율을 이미 보유. 모든 새 개념은 이 넷 중 하나에 앵커한다.
- **수식은 항상 3형태로**: ① 직관 문장 ② 수식 ③ 실행 torch 코드 (2.13.0+cu126 설치 완료, RTX 4070).
- **검증**: 각 모듈 끝에 "손계산 = 코드 출력" 일치 확인 문제. NYPC 규율(작은 데이터 overfit 먼저, holdout)을 수학 자체에도 적용.
